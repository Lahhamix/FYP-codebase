The executive summary should be one page or less. The purpose of this summary is to give a snapshot of the overall structure of the final year project and the main obtained results.\\ ~\\ Accordingly, you should include the following:\\ ~\\ 
1- A summary about the motivation for your project \\
2- What is the problem that is being solved? \\
3- A brief summary about the proposed solution\\
4- A brief summary about the main findings of the project
